# Security Policy

## Supported Versions

The main branch is actively maintained and supported.

## Reporting a Security Issue

If you discover a security issue within this project:

1. Open a GitHub issue describing the problem.
2. Include the affected phase(s).
3. Include steps to reproduce the behavior.
4. Include expected and actual results.

This repository is a cybersecurity simulation and educational platform and is not intended for production deployment.

## Scope

This project demonstrates:

* Zero Trust concepts
* Detection engineering
* Threat hunting
* SOAR workflows
* Cloud security monitoring
* Identity security
* AI-assisted SOC operations

Any findings related to these components are welcome.
